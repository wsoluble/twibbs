ツイコロとは

Twitterで流れているユニークなツイートをピックアップして、みんなで話を広げて楽しむことを目的とした、Twitterユーザーのためのコミュニティです。

コンペとは

コンペとは、コンペティション(competition)の略で競争、競技会の意味です。ツイコロでは、東西南北の一連のツイートへの反応の多さを「勢力」として表現します。

ツイートの批判を目的とした機能はありません

勢力の算出方法

勢力＝ひとことポイント＋便乗ポイント＋座布団ポイント

  • ひとことポイント：ツイートへのひとことコメント（250pt／1件）
  • 便乗ポイント：ひとことコメントへの便乗コメント（500pt／1件）
  • 座布団ポイント：ツイ士へ投げられた座布団（50pt／1件）

「ひとこと」や「そうだよ」は無制限にできるの？

ひとことコメント、便乗コメント、座布団は、１つのコンペにつきそれぞれ１回だけ投稿することができます。

投稿するためには当サイトへのTwitterアカウントを使ったログインが必要となります

コンペやコメントを投稿しても反映されない場合

数秒お待ち頂ければ大抵は反映されます。

サーバーの状態によっては数分かかることがあります。

対応ブラウザ

当サイトは、最新のブラウザに対応しています。InternetExplorerは10以上で閲覧してください。

InternetExplorer9以下は単色表現のみ対応しており角ばった外観となります。IE7以下はレイアウトが崩壊します。

不適切な書き込みを見かけたときは

  • 当サイトはマイページを設けており、投稿者がコメントやコンペを非公開にできます
  • 禁止事項(利用規約にて記載)を含むコメントやコンペの非公開依頼は、まずはTwitter等を通じて投稿者へ問い合わせてください
  • 投稿者のTwitterアカウントが消失するなどしてコンタクトがとれない場合は、下記のフォームより当サイト管理宛に非公開依頼を送信してください
  • 誰について言及しているのか不明な場合や、実在しない人物へ言及するコメントは非公開にならない場合があります
  • 非公開処理完了までに数日かかることがあります

コメントやコンペには投稿者のTwitterIDが表示されています

非公開依頼・不具合報告

  • 禁止事項を含む不適切な書き込み、または不具合など発見しましたらこちらのフォームよりご報告ください。
